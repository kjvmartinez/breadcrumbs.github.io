HTML Introduction

Learning Objectives

  1. Understand the basic structure of an HTML document, including the <!DOCTYPE>, <html>, <head>, and <body> elements.
  2. Create paragraphs using the <p> tag and apply different levels of headings using <h1> to <h6> tags.
  3. Create hyperlinks using the <a> tag to connect to other web pages or resources.
  4. Create both ordered (numbered) and unordered (bulleted) lists using the <ul>, <ol>, and <li> tags.
  5. Insert images using the <img> tag and specify image sources and alt attributes.
  6. Apply basic formatting using tags like <strong>, <em>, and <br> for bold, italic, and line breaks.

HTML stands for HypterText Markup Language. HypterText defines the links betweet web pages. Markup language is used to define the text within the tag which define the structure of web pages.

HTML is a markup language used by the browser to manipulate text, images, and other content, in order to display it in the required format. HTML was created by Tim Berners-Lee in 1991. The first-ever version of HTML was HTML 1.0, but the first standard version was HTML 2.0, published in 1995.

HTML uses tags and elements that tells the browswer how to display the content. Remember to always include the closing tag. If ommitted the browser will apply the opening tag until the end of page.
