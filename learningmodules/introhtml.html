HTML Introduction

Learning Outcomes

  1. Understand the basic structure of an HTML document, including the <!DOCTYPE>, <html>, <head>, and <body> elements.
  2. Create paragraphs using the <p> tag and apply different levels of headings using <h1> to <h6> tags.
  3. Create hyperlinks using the <a> tag to connect to other web pages or resources.
  4. Create both ordered (numbered) and unordered (bulleted) lists using the <ul>, <ol>, and <li> tags.
  5. Insert images using the <img> tag and specify image sources and alt attributes.
  6. Apply basic formatting using tags like <strong>, <em>, and <br> for bold, italic, and line breaks.

HTML stands for HyperText Markup Language. HypterText defines the links between web pages. Markup language is used to define the text within the tag, which defines the structure of web pages.

HTML is a markup language used by the browser to manipulate text, images, and other content in order to display it in the required format. HTML was created by Tim Berners-Lee in 1991. The first-ever version of HTML was HTML 1.0, but the first standard version was HTML 2.0, published in 1995.

HTML uses tags and elements that tells the browswer how to display the content. Remember to always include the closing tag. If ommitted the browser will apply the opening tag until the end of page.

Tags and Elements
HTML Basic Structure

    <!DOCTYPE html>
    <html>
        <head>
            <title>About</title>
        </head>
        <body>
            <h2> History </h2>
            <p> It started as a weekend project... </p>
        </body>
    </html>

Here's a list of the HTML tags used in the code above.

  • <!DOCTYPE html>: Specifies the document type and version of HTML being used.
  • <html>: The root element that wraps around the entire HTML document.
  • <head>: Contains meta-information about the document, like character encoding and linked resources.
  • <title>: Sets the title of the web page, which appears in the browser's title bar or tab.
  • <body>: Contains the visible content of the web page.
  • <h2>: Headings, used to define different levels of headings for sections.
  • <p>: Represents a paragraph of text.

Anchor Tag <a>

The <a> tag in HTML is the anchor tag, commonly known as the hyperlink tag. It's used to create hyperlinks that link to other web pages, files, or resources, both within the same website and to external websites. The <a> tag is one of the fundamental elements of HTML for creating navigation and connecting different web pages together.

The basic syntax of the <a> tag is as follows:


    <a href="URL">Link Text</a>
  • href: This attribute specifies the URL (Uniform Resource Locator) to which the link points. It can be an absolute URL (e.g., https://www.example.com) or a relative URL (e.g., about.html).
  • Link Text: This is the text that is displayed on the webpage for the user to click on.

You can use the <a> tag to create internal links that direct users to specific sections within the same page by utilizing the href="#section-id" syntax.


    <a href="#section-about">Jump to About Section</a>
    ...
    <section id="section-about">
        <h2>About Us</h2>
        <p>This is information about our organization...</p>
    </section>

Different types of Hyperlinks


    <!-- External Website Link -->
    <p><a href="https://www.example.com">Visit Example.com</a></p>
    
    <!-- Internal Page Link -->
    <p><a href="about.html">Learn About Us</a></p>
    
    <!-- Email Link -->
    <p><a href="mailto:info@example.com">Contact Us</a></p>
    
    <!-- Downloadable File Link -->
    <p><a href="documents/sample.pdf" download>Download PDF</a></p>

Unordered list <ul> </ul>

An unordered list is a list of items where each item is preceded by a bullet point. You can create an unordered list using the <ul> element and list item <li> elements. 


    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>

Output of the code above:

  • Item 1
  • Item 2
  • Item 3
Ordered list <ol> </ol>

An ordered list is a list of items where each item is preceded by a number or letter. You can create an ordered list using the <ol> element and list item <li> elements. 


    <ol>
        <li>First item</li>
        <li>Second item</li>
        <li>Third item</li>
    </ol>

Output of the code above:

  1. First item
  2. Second item
  3. Third item
Nested Lists

You can also create nested lists by placing one type of list inside another. For example, you can have an ordered list with unordered lists as its items, or vice versa. 


    <ol>
    <li>Main point 1</li>
    <li>Main point 2
        <ul>
            <li>Subpoint 1</li>
            <li>Subpoint 2</li>
        </ul>
    </li>
    <li>Main point 3</li>
    </ol>

Output of the code above:

  1. Main point 1
  2. Main point 2
    • Subpoint 1
    • Subpoint 2
  3. Main point 3

Image Tag <img>

The <img> tag in HTML is used to embed images into a webpage. It allows you to display images such as photographs, illustrations, icons, and logos on your web pages. Here's how you can use the <img> tag: 


    <img src="image-source.jpg" alt="Image Description">
  • src: This attribute specifies the source URL of the image. It can be an absolute URL (e.g., https://www.example.com/image.jpg) or a relative URL (e.g., images/image.jpg), indicating the location of the image file.
  • alt: This attribute provides alternative text for the image. It's important for accessibility and search engine optimization. The text is displayed if the image cannot be loaded or for users with visual impairments.

This is an example of embedding an image using the img tag.


    <img src="image-example.jpg" alt="A beautiful landscape">

<strong> Tag

The <strong> tag is used to indicate that its enclosed text should be given strong importance. By default, it often results in bold text, but its primary purpose is to convey semantic emphasis rather than just visual styling. 


    <p>This is a <strong>very important</strong> point.</p>
<em> Tag

The <em> tag is used to emphasize text, typically resulting in italics. Like the <strong> tag, its primary purpose is to convey semantic emphasis. 


    <p>This sentence contains an <em>emphasized</em> word.</p>
<br> Tag

The <br> tag is used to insert a line break within text content. It doesn't have a closing tag, and it's commonly used to force a new line within a paragraph or other inline elements. 


    <p>This is the first line.<br>
    This is the second line.</p>
